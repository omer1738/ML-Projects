# Medical Insurance Cost Prediction

A machine learning project that predicts medical insurance charges using **Linear Regression**.

##  Dataset
The dataset contains **1338 records** with the following features:
- Age
- BMI
- Number of children
- Smoker status
- Region
- Sex

##  Model
- **Algorithm:** Linear Regression (scikit-learn)
- **Target:** Insurance charges (USD)
- **Input features used:** Age, BMI, Children, Smoker status

##  Libraries Used
- pandas
- numpy
- scikit-learn
