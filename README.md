### 3.  Decision Trees & Random Forests
**File:** `DescisionTrees.ipynb`

Applies tree-based models to a classification problem, with deep exploration of hyperparameter tuning.

**What's covered:**
- Data loading and preprocessing
- Imputing missing values
- Feature scaling and encoding
- Training a Random Forest classifier
- Hyperparameter tuning:
  - `max_depth` and `max_leaf_nodes`
  - `min_samples_split` and `min_samples_leaf`
  - `min_impurity_decrease`
  - `bootstrap` and `max_samples`
- Making predictions on new inputs

##  How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/omer1738/ML-Projects.git
   cd ML-Projects
   ```

2. Install dependencies:
   ```bash
   pip install scikit-learn pandas numpy matplotlib seaborn jupyter
   ```

3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file and run the cells.
